# Ower AI

智能多模态对话平台，基于 **ReAct 模式**（感知→思考→决策→行动）驱动，自动路由文本 / 图片 / 视频模型。

## 特性

- **ReAct 智能路由** — 先调 LLM 分析用户意图，再决策调用文本 / 图片 / 视频模型
- **多模态生成** — 文本对话、图片生成、视频生成
- **加密配置** — API Key 和端点经过 XOR + Base64 加密，不在源码中暴露明文
- **自适应 UI** — 暗色赛博风格，桌面三栏 / 移动端全屏自适应
- **缩略图预览** — 图片缩略图展示，点击弹窗查看

## 使用

直接用浏览器打开 `index.html` 即可，无需构建步骤。

项目内置了 Agnes API 的调用配置，开箱即用。

## 项目结构

```
├── index.html       # 主页面
├── css/style.css    # 样式
├── js/
│   ├── app.js       # 主逻辑（ReAct 路由、消息渲染）
│   ├── api.js       # API 调用（模块版）
│   ├── config.js    # 配置加载（模块版）
│   └── storage.js   # 文件下载
└── config.json      # API 配置
```
